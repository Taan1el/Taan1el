## Start here

Full-stack developer from Estonia building local-first AI tools, Obsidian
workflows, and interactive web apps. Below is my selected work, each with a live
demo or source and what to look at. Quickest read: the portfolio OS demo.

- Portfolio: https://taan1el.github.io/taaniel-portfolio-os/
- Email: Taaniel.vananurm@gmail.com

## Selected work

### [taaniel-portfolio-os](https://github.com/Taan1el/taaniel-portfolio-os)
An immersive, browser-based desktop OS that doubles as my portfolio.
[Live demo](https://taan1el.github.io/taaniel-portfolio-os/). TypeScript and Vue.
Look at: the window manager and app shell.

### [cortex](https://github.com/Taan1el/cortex)
A local AI side panel for Obsidian. Local-first, so it works against your notes
without a cloud round trip.
Look at: the Obsidian integration and the local model wiring.

### [save-to-obsidian](https://github.com/Taan1el/save-to-obsidian)
Saves ChatGPT conversations straight into local Obsidian notes, with a browser
extension plus a localhost helper.
Look at: the extension to localhost bridge.

### STRATA (in progress, private)
An original single-player voxel survival game in Godot 4.6 and C#. Deterministic
world generation, layered biomes, living mobs, and an engine-free rules layer
with hundreds of unit tests. Source available on request.
